# Career Mapping Workshop — Facilitator Guide

## Session Overview

**Format:** Interactive workshop (90 min)
**Audience:** Early-career BAs and career changers
**Goal:** Help participants map their current skills to the BA competency framework and identify targeted growth paths.

## Learning Objectives

By the end of this session, participants will be able to:
- Locate their current experience on the BABOK competency spectrum
- Identify 2–3 high-priority skill gaps relative to their target role
- Draft a 90-day action plan with concrete next steps

- [ ] Career Mapping worksheet (participant_handout.md)

## Agenda

| Time | Segment | Notes |
|------|---------|-------|
| 0:00 | Welcome + framing | Set context: this is a mirror, not a test |
| 0:10 | Self-assessment activity | Participants rate 6 competency areas 1–5 |
| 0:25 | Small group sharing | Pairs compare maps, note patterns |
| 0:40 | Gap analysis framework | Facilitator walks through: current → target → gap |
| 0:60 | Action planning | Participants draft 90-day plan |
| 0:80 | Share-outs + commitments | 2–3 volunteers share one commitment |
| 0:85 | Wrap-up + next steps | Point to resources, announce next session |

## Facilitation Notes

### Opening (10 min)
[FACILITATOR: Start with a brief personal story about a career transition or unexpected BA opportunity. This builds psychological safety before self-assessment.]

Use the framing: "There's no right answer on this map — only useful information."

### Self-Assessment Activity (15 min)
[FACILITATOR: Have participants rate themselves on: Requirements elicitation, Stakeholder communication, Process modeling, Data analysis, Solution scoping, Documentation. Use 1 = no exposure, 5 = can teach this.]

Watch for under-raters (common in underrepresented groups) — normalize that a 3 is solid.

### Gap Analysis Framework (20 min)
[FACILITATOR: Draw the bridge diagram: Current State → Gap → Target State. Emphasize that gap size isn't the problem; unclear direction is.]

Key questions to pose:
- "What does your next role ACTUALLY require day 1?"
- "Which gap, if closed, would have the highest leverage?"
- "What are you already doing that you're not counting?"

### Action Planning (20 min)
[FACILITATOR: Circulate during planning. Listen for vague plans ("I'll study more") and help sharpen them ("Which BABOK chapter? By when? How will you know you've got it?")]

## Debrief Questions

- What surprised you about your own map?
- Where did your self-rating differ from how others see you?
- What's the one action you can take this week?

## Common Facilitation Pitfalls

- **Rushing the self-assessment** — give people the full 15 min; resist filling silence
- **Focusing only on gaps** — celebrate what's already strong
- **Action plans that are too broad** — push for specificity (what, when, how to measure)
