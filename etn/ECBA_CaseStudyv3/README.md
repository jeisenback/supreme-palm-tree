# ECBA Case Study — Format v3: Bi-Weekly, Without Session B

--- DOCUMENT GUIDE ---
What this is:   Format variant of the TrailBlaze Adventures ECBA Case Study series. Describes how to run the 5-session program on a bi-weekly schedule with Session A only — no SME deep dives.
Who it's for:   Facilitator; program coordinator; board approver.
When to use it: When the chapter cannot reliably staff 5 SME speakers, when participants can commit bi-weekly but not weekly, or when a 10-week A-only track is preferable to the full 10-session v2 format.
Relationship:   Content is identical to the base series (etn/ECBA_CaseStudy/). Session B content is removed. Schedule, pacing, and homework expectations differ.
Restricted:     No — shareable with board and interested participants.
--- END GUIDE ---

## What Changes in This Format

| Factor | Base Series (v1 — Monthly + B) | This Format (v3 — Bi-Weekly, A Only) |
|--------|-------------------------------|--------------------------------------|
| Cadence | A sessions monthly; B sessions mid-month | A sessions every 2 weeks; no B sessions |
| Overall duration | ~5 months | ~10 weeks |
| Total sessions | 10 (5A + 5B) | 5 (5A only) |
| Time between A sessions | 4 weeks | 2 weeks |
| SME involvement | Required (5 speakers) | None |
| Homework window | ~3 weeks per cycle | ~2 weeks per cycle |
| Session B stretch question | Feeds next A share-out | Not used — replaced with extended homework prompt |
| Who it fits | Working professionals with flexible schedules | Cohorts with constrained SME access or participants who want a focused, leaner commitment |

## What Does Not Change

- TrailBlaze Adventures case study, session-gated reveals, and all in-session exercises are identical.
- All Session A artifacts (BACCM chart, stakeholder map, requirements, RTM, solution recommendation) are the same.
- Session A is 90 minutes.
- ECBA domain coverage and BABOK chapter mapping are unchanged.
- All Session A facilitator guides in `etn/ECBA_CaseStudy/` apply directly.

## What Is Removed

- All five Session B sessions are removed.
- SME recruitment is not required.
- Session B stretch questions are not used (replaced with extended homework prompts — see plan).
- The Session B deep-dive friction (mock interviews, requirements challenges, CEO pushback) is not replicated in this format.

## Design Implication: What Session B Was Doing

Session B served a specific function: it added real-world friction to participants' in-session work at the moment when they had begun to feel confident about it. Without Session B, participants do not get that friction surface. The facilitator must compensate:

1. **Extended homework prompts** — each v3 homework includes one additional task that approximates the Session B challenge. These are included in the plan.
2. **Sharper share-outs** — the facilitator should probe harder in share-outs (Sessions 2A–5A) because the B-session questions that would have surfaced wrong assumptions are not present. Name common misconceptions before the share-out, not after.
3. **Checkpoint density** — add one additional checkpoint question per session if the group appears to have skated through the exercise. The plan marks these as optional checkpoints.

## When to Use This Format

Use v3 when:
- SME recruitment is a constraint — no guaranteed speaker pipeline.
- Participants cannot commit weekly but can commit bi-weekly.
- The goal is to complete the ECBA content in ~10 weeks without the full SME program.
- You are piloting the series for the first time and want a simpler logistics footprint.

Consider v2 instead when participants have confirmed weekly availability and the chapter has an SME network to draw from.
